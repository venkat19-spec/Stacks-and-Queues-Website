Stacks

What is a Stack?

Stack is a linear data structure which follows a particular order in which the operations are performed. The order may be LIFO(Last In First Out) or FILO(First In Last Out).

How can we implement a Stack?

A stack can be implemented in the following ways: Array and Linked List.

Array

Structure of the stack is as follows:

                    Structure stack:
                              maxsize : integer
                              top : integer
                              items : array of item

Intilialising the stack:

                        procedure initialize(stk : stack, size : integer):
                              stk.items ← new array of size items, initially empty
                              stk.maxsize ← size
                              stk.top ← 0

Pushing:    Adds an element, increments top after checking overflow

                        procedure  push(stk : stack, x : item):
                           if  stk.top = stk.maxsize:
                              report overflow error
                           else
                              stk.items[stk.top] ← x
                              stk.top ← stk.top + 1

Popping:    Decrements top after checking for underflow , then returns the element at previous top 

                        procedure  pop(stk : stack, x : item):
                           if  stk.top = 0:
                              report underflow error
                           else
                              stk.top ← stk.top - 1
                              r ← stk.items[stk.top]
                              return r
Linked List

Another option to implement stacks. A stack is then a pointer to the "head" of the list, with perhaps a counter to keep track of the size of the list:

                        Structure node:
                                  data : item
                                  next : node or nil
                    
                        Structure stack:
                                  head : node or nil
                                  size : integer
                    
                        procedure   initialize(stk : stack):
                              stk.head ← nil
                              stk.size ← 0

Pushing and Popping happens at the head, so no overflow can happen but underflow can happen

                        procedure push(stk : stack, x : item):
                              newhead ← new node
                              newhead.data ← x
                              newhead.next ← stk.head
                              stk.head ← newhead
                              stk.size ← stk.size + 1                    
                    
                        procedure pop(stk : stack):
                           if  stk.head = nil:
                              report underflow error
                           r ← stk.head.data
                           stk.head ← stk.head.next
                           stk.size ← stk.size - 1
                           return r
Table of Operations
Operation Syntax Result
Push P x, x is a number Pushes an element onto the stack
Pop PO Removes the top element from the stack
Top T Returns top element of the stack
Is Empty I Returns a boolean value if stack is empty or not
Is Full (only for Array impl.) F Returns a boolean value if stack is empty or not
Length L Returns size of the stack
